# Question Draft: 議員定数と議会改革

## Context

本市議会においても、定数削減の議論が避けて通れない時期に来ています。しかし、単なる「数合わせ」の削減は、議会の自殺行為になりかねません。

## Question 1 (To: 議会事務局長)

過去3回の選挙における立候補者倍率の推移と、定数割れのリスク試算について伺います。

## Question 2 (To: 市長/他会派)

他市事例によれば、定数削減のみを行った議会で、かえって無投票当選が増えたケースがあります。この点について、コスト削減以外の「機能維持」の観点からどう考えますか。

## Question 3 (Proposal)

単なる定数削減ではなく、報酬体系の見直しや夜間議会の導入など、「なり手を増やす」ための包括的な議会改革プランを策定すべきと考えますが、見解を伺います。
