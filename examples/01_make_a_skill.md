# Make a Skill

## Skill Request Card

Copy this into your prompt:

```markdown
I need a new skill.

- **Name**:
- **Goal**:
- **Draft Logic**:
```

## Procedure

1.  Run `skill_scout_web` to ensure uniqueness.
2.  Run `universal_skill_factory` with the card above.
3.  The factory will generate `skills/<name>/SKILL.md`.
