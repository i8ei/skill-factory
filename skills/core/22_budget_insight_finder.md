# 22_budget_insight_finder — 予算分析 (v0.1)

## 役割

正規化された予算データから、「増減がおかしい」「説明不足」「前例踏襲」などの異常値を検知する。

## 入力

- 21で正規化された予算データ
- 過去数年のデータ（比較用）

## 出力（固定フォーマット）

### Insight Report

- Significant Changes: (昨年比±10%以上など)
- New Projects: (新規事業)
- Stagnant Projects: (長年成果が見えない事業)
- Anomaly Flag: (桁がおかしい、説明と金額が合わない等)

## 絶対ルール

- 事実（増減）と推測（無駄遣いかも）を明確に分ける。

## よくある失敗

- 金額が小さいが増減率が高いだけの項目に時間を使ってしまう。
