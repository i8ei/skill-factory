# 50_protocol_converter — 議会プロトコル変換 (v0.1)

## 役割

一般的な文章やアイデアを、議会（本会議、委員会）で通じる「独特の言い回し・作法」に変換する。

## 入力

- 質問したい内容（骨子）
- エビデンス（11）

## 出力（固定フォーマット）

### Parliamentary Script (Draft)

- Context: (まくら言葉、導入)
- Question 1: (事実確認)
- Question 2: (見解を問う)
- Question 3: (提案・要望)
- Closing: (結び)

## 絶対ルール

- 通告制か、一問一答かなどの「ルール」を確認する（Adapter領域だが、意識はする）。
- 品位を欠く表現（罵倒など）を排除する。

## よくある失敗

- 質問が長すぎて、何を聞いているのかわからなくなる（一問一答の原則を守る）。
